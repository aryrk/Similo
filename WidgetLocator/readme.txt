Replication package for Similo experiment
-----------------------------------------

Requires: Java 8 or newer.

Select the application to test by editing the RunWidgetLocator.bat file. Set the application number between 0 and 47 (48 apps).

Start experiment by double-clicking the RunWidgetLocator.bat file.

The result is stored in the WidgetLocatorResultsTable.txt and Performance.txt files.

The apps folder contains all target and candidate locator parameters (the target XPath) for all websites.

The widgetlocator folder contains the Java source code.
